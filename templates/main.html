Le projet PPII-2 (Projet Pluridisciplinaire d'Informatique Intégrative) a pour but de concevoir une application similaire au jeu WORDLE, ainsi qu'un solveur capable de jouer à WORDLE.
WORDLE est un jeu semblable au célèbre MOTUS, animation télévisée diffusée de 1990 à 2019 en France.

Des groupes de 4 ont ainsi été constitués dans la promo 2024 afin de réaliser cette application.
Les conditions du projet sont telles que l'application doit regrouper et faire interagir une interface web, une base de données, et utiliser Flask / Python comme pour le PPII-1.

TelecomNancy est une grande école d'ingénieurs en informatique et sciences du numérique (cybersécurité, big data, intelligence articielle, internet des objets...).

TelecoMotus, un remake du WORDLE

Le jeu WORDLE, semblable au MOTUS, a pour but de trouver un mot-solution prédéfini en soumettant tour à tour différents mots avec le même nombre de lettres.
Le mode de jeu le plus connu, celui du WORDLE officiel, offre 6 essais pour trouver un mot de 5 lettres. Vous pouvez directement accéder à ce mode en cliquant sur le lien ci-dessous.

Wordle Classic

Le mode de jeu pour les joueurs confirmés. Trouvez les mots proposés dans un temps limité. A chaque mot trouvé, vous gagnez un bonus de temps. La partie s'arrête quand le compteur arrive à 0.
